Retrieves a list of ads, possibly filtered.

A builder for the list method supported by a ad resource. It is not used directly, but through a AdMethodsBuilder.

Example

Instantiate a resource method builder

 
// You can configure optional parameters by calling the respective setters at will, and
// execute the final call using `doit()`.
// Values shown here are possibly random and not representative !
let result = hub.ads().list("profileId")
             .type_("diam")
             .ssl_required(true)
             .ssl_compliant(true)
             .sort_order("amet.")
             .sort_field("ipsum")
             .size_ids("ut")
             .search_string("dolor")
             .remarketing_list_ids("sea")
             .placement_ids("ut")
             .page_token("eirmod")
             .overridden_event_tag_id("sanctus")
             .max_results(-27)
             .landing_page_ids("dolor")
             .ids("et")
             .dynamic_click_tracker(false)
             .creative_type("vero")
             .creative_optimization_configuration_ids("ut")
             .creative_ids("sed")
             .compatibility("et")
             .campaign_ids("ipsum")
             .audience_segment_ids("justo")
             .archived(true)
             .advertiser_id("vero")
             .active(false)
             .doit();
