Create an iterator running multiple iterators in lockstep.

The iterator Zip<(I, J, ..., M)> is formed from a tuple of iterators and yields elements until any of the subiterators yields None.

Iterator element type is like (A, B, ..., E) where A to E are the respective subiterator types.

Example

use itertools::Zip;

// Iterate over three sequences side-by-side
let mut xs = [0, 0, 0];
let ys = [69, 107, 101];

for (i, a, b) in Zip::new((0i32..100, xs.iter_mut(), ys.iter())) {
   *a = i ^ *b;
}

assert_eq!(xs, [69, 106, 103]);
