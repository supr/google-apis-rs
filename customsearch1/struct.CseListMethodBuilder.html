Returns metadata about the search performed, metadata about the custom search engine used for the search, and the search results.

A builder for the list method supported by a cse resource. It is not used directly, but through a CseMethodsBuilder.

Example

Instantiate a resource method builder

 
// You can configure optional parameters by calling the respective setters at will, and
// execute the final call using `doit()`.
// Values shown here are possibly random and not representative !
let result = hub.cse().list("q")
             .start(6)
             .sort("rebum.")
             .site_search_filter("dolore")
             .site_search("nonumy")
             .search_type("sed")
             .safe("aliquyam")
             .rights("sit")
             .related_site("eirmod")
             .or_terms("consetetur")
             .num(16)
             .lr("sed")
             .low_range("ea")
             .link_site("gubergren")
             .img_type("aliquyam")
             .img_size("eos")
             .img_dominant_color("tempor")
             .img_color_type("sea")
             .hq("labore")
             .hl("ipsum")
             .high_range("aliquyam")
             .googlehost("dolores")
             .gl("sit")
             .filter("diam")
             .file_type("ut")
             .exclude_terms("justo")
             .exact_terms("est")
             .date_restrict("amet")
             .cx("accusam")
             .cref("clita")
             .cr("diam")
             .c2coff("justo")
             .doit();
